P3

Website for a Future Event

Choose a future event. This could be an already-existing future event such as one within the Yale community. Or, this future even could be completely fictional altogether.

Phase 1: Artifacts

After choosing a future event, bring in at least 5 “visual artifacts” related to the event. Some examples include...

  • — flyer or poster about event
  • — event invitation with envelope
  • — brochure of event agenda (if complex and spanning multiple days)
  • — Google Map of event location(s)
  • — event’s Facebook page
  • — email thread between event collaborators (with attachments?)
  • — You can find a more expansive list of possible artifacts below

Each artifact should give some sense or glimpse into the event’s purpose and possible narrative. Think about the event over time, as these visual artifacts can be from all phases: the event planning, the event itself, and the event’s aftermath. You are imagining and creating these artifacts to begin wrapping your mind around your event and what it’s about. This is a preliminary phase; but please try to be as specific and detail-oriented as possible while creating your artifacts. These artifacts should not paint a complete picture; instead, they should make us want to know more about your event.

If your event is already-existing (a “real” event), you can bring in some found artifacts. (For example: flyer, Facebook event page, etc.) You should still create 5 artifacts, but your found artifacts could possibly inform or guide your process.

To complete this phase, present documentation of your 5 visual artifacts on a simple website linked to your class homepage.

Phase 2: Event Website

Make a website providing information about your event. In this phase, focus on giving your event a distinct visual identity and language specific to your event.

Your website should include...

  • — the title of the event
  • — the time and date of the event
  • — the location of the event
  • — a map of the event’s location(s)
  • — a trace of the passing of time (in some way)
  • — a piece of text (min 500 words) about or related to the event
  • — a navigation to view all these pieces

During this phase, please plan to produce one of your artifacts physically. A list of websites that produce physical artifacts from digital files is included below.

Phase 3: Event Website as Time Capsule

Make either a separate website or an addendum to your current event website showing your event as you would like it perceived now that is over.

In this website, include artifacts that are important for future visitors of your website to understand and reference your event. These artifacts could include the artifacts imagined and created in Phase 1, but please feel free to go further now that you have a better mental picture of your event. Your website should also include documentation of your physical artifact.

Beginning (event planning and advertisement)

  • — flyer or poster about event
  • — event invitation with envelope
  • — brochure or PDF of event agenda (if complex, spanning multiple days)
  • — designed item of clothing worn exclusively at event
  • — other designed object(s) given out at event
  • — screenshot from event iPhone app
  • — Google Map of event location(s)
  • — Google Doc of directions to the event
  • — Google Doc of inclement or severe weather plans (if event is outside)
  • — event’s Facebook page (showing people who RSVP’d)
  • — email thread between friends about event (with attachments?)
  • — email thread between event collaborators (with attachments?)

Middle (event itself)

  • — audio recording of event
  • — video recording of event
  • — image recording of event
  • — text threads between event participants
  • — tweets with special event hashtag
  • — altered physical objects (somehow transformed during event)

End (event aftermath)

  • — curated selection of audio, video, or image recording of event
  • — summary of event on blog or magazine website
  • — event’s Facebook page (showing comments from participants)
  • — event’s Yelp page (showing reviews from participants)

Websites that produce physical artifacts from digital files

  • — Lulu.com (books, calendars, other printed material)
  • — Spoonflower.com (fabric)
  • — Zazzle.com (all sorts of useful printable items, like tshirts)
  • — Shapeways.com (3D printed objects)

In conclusion

This project is about visual identity over time. You’ll do this by first by creating 5 visual artifacts. Then, once the event is better formed in your mind, you’ll create a cohesive website for the event, focusing on giving your event a visual identity to transmit the event’s purpose and spirit. Finally, you’ll create an event time capsule — fit for the future.
